# dust

code generation system
Single Page Application Isomorphic Example for Egg + Vue, Front-End and Node of The Application are Written in TypeScript.


## 安装依赖

```bash
npm install
```

## 本地启用

```bash
$ npm run dev
$ open http://localhost:7001
```

## 打包构建

```bash
npm run tsc
npm run build
npm start
```
